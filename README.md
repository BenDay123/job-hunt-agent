# Job Hunt Agent

An agent that finds Fortune 250 job listings matching my skills, drafts tailored
application materials, and gets them in early — while keeping me in the loop on
the final submit.

Build-in-public. Successor to
[workday-autofill-agent](../workday-autofill-agent), which becomes the submission
layer for Workday-hosted roles.
